Expire chats
This extension will delete chats whose last message is older than the specified number of days. The currently open chat will not be deleted.
If enabled, chat backups older than the specified number of days will also be deleted. If disabled, only chats will be deleted, not backups.
If enabled, chats and (if enabled) backups will be automatically deleted on page load without confirmation. Only a notification will be shown with the results.
⚠️ Warning: Deletion is permanent and cannot be undone.
